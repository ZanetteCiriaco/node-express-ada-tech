# node-express-ada-tech

Projeto do módulo node express do curso da Ada tech, trilha backend.

## Instruções de Configuração

1. Criar arquivo `.env`

   Exemplo (preencher os campos vazios):

   ```plaintext
   PORT=
   DB_USER='postgres'
   DB_HOST='db'
   DB_NAME='postgres_db'
   DB_PASSWORD=
   DB_PORT=

   JWT_SECRET=
   JWT_EXPIRESIN=

    
## Executar:
    npm install
    docker-compose build
    docker-compose
